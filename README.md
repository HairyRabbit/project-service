# ProjectService

A backend project template for TypeScript/GraphQL/ExpressJS/Apollo


## Project Struct 

```txt
|- <root>/
  |- graphql/ # source code for gql type defintions and resolvers
  |- service/ # restful code 
  |- init/    # init script exec before server start
  |- app.ts   # entry file
  |- .env     # env defintion
```
